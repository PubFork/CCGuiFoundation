
# CCGuiFoundation (*C++*)
===========================

***CCGuiFoundation*** is a simplified GUI framework using STL and WTF. Developed by ***bajdcc***.
*PS.* Most of codes refer to ***GacLib*** developed by [*vczh*](https://github.com/vczh "Github page of vczh").
